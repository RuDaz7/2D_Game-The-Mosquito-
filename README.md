# 모기 헌터_Mosquito Hunter
+ ## 개임소개
> ### 무더운 여름 날 모기로 밤잠을 설치던 한 남자
> ### 꿈속에서 그토록 자신을 괴롭히던 모기를 사냥할 수 있게 된다.  
---  
+ ## 게임방법  
> ### **짜증나는 모기를 잡으며 강해지자!**
>  모기에게서 오래 살아남을 수록 **고혈압 게이지 상승**
>  모기를 잡을 수록 **Coin Get**
>  고혈압 게이지로 "Skill" 습득 가능  
>  Coin으로 각종 아이템 구매 가능
